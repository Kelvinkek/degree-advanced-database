# Advanced Database Management — Airline System (Oracle PL/SQL)

A comprehensive Oracle PL/SQL implementation of an **Airline Management System**, covering schema design, data population, stored procedures, triggers, analytical queries, and business reports.

This project was completed as part of the Advanced Database Management module during our degree.

## Project Overview

The system models a real-world airline business with 18 entities including passengers, memberships, flights, pilots, airplanes, bookings, payments, refunds, cancellations, luggage tracking, and aircraft maintenance.

It demonstrates advanced PL/SQL features:
- **Identity columns** with `GENERATED BY DEFAULT ON NULL AS IDENTITY`
- **Dynamic SQL** via `EXECUTE IMMEDIATE` for sequence management
- **Exception handling** with `EXCEPTION WHEN OTHERS`
- **Constraints** including check constraints with `REGEXP_LIKE` for email/phone validation
- **Stored procedures** with parameter validation and transactional logic
- **Triggers** for automated business rules (loyalty points, promotions, status updates)
- **Complex analytical reports** with aggregations, joins, and window functions

## How to Run

This project targets **Oracle Database** (use SQL Developer, SQL*Plus, or any Oracle client).

1. **Build the schema**:
   ```sql
   @ADM_DDL.txt
   ```
2. **Load sample data**:
   ```sql
   @ADM_DML.txt
   ```
3. **Set up sequences** (auto-syncs `START WITH` to existing max IDs):
   ```sql
   @run_sequence.txt
   ```
4. **Install database objects** — execute the `.sql` / `.txt` files inside each member's folder under `code/` to create their procedures, triggers, and reports.
5. **Run reports / queries** — execute any individual report or query script.
